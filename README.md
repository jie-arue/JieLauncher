**OCR機能搭載・タイマー＆暗フィルター付きフローティング・ランチャー**

複数のゲームのスタミナ待ちを忘れないためのAndroid用ランチャーアプリです。

## 🌟 主な特徴

### 1. 常駐型フローティングメニュー
画面の最前面に常に表示される小さなメニューから、お気に入りアプリ（最大18個）をワンタップで起動。
- **コンパクトモード**: 邪魔な時は折りたたんで「日付」「時刻」「電池残量」「電池温度」を表示するステータスモニターとして機能します。
- **自由な配置**: ドラッグ＆ドロップで画面上の好きな場所に配置可能。

### 2. 視覚的な制限「暗（ダーク）フィルター」
アプリ使用後、アイコンが暗く（グレーアウト）なり、クリックを制限するクールダウン機能を搭載。
- **分指定**: 「起動後60分間」などの時限ロック。
- **時刻指定**: 「18:00まで」など、特定の時刻までのロック。
- **日替わり解除**: 未設定時は「翌日深夜0時」に自動解除。1日1回ルールの運用に最適。

### 3. 【独自機能】自動タイマー (OCR)
Google ML Kitを利用し、画面上の数字を自動で読み取ってタイマーをセットします。
- **ワンタッチ設定**: ゲーム内のスタミナ回復待ち時間などを画面キャプチャで読み取り、手入力なしで正確なタイマーをセット。
- **高精度ロジック**: 読み取り失敗時に15秒以内の再試行で精度を上げるリカバリ機能を搭載。秒数は切り上げて「+1分」でセットされるため、確実に解除を待てます。

### 4. 省電力設計
一定時間操作がない場合、UIの更新頻度を自動で下げる機能を搭載し、バッテリー消費を抑えます。

## 🛠 セットアップ
本アプリの動作には以下の権限が必要です：
1. **他のアプリの上に重ねて表示**: メニューの常駐に必要。
2. **バッテリー最適化の無視**: バックグラウンドでのタイマー動作に必要。
3. **通知の送信**: システムによる強制終了防止。
4. **画面キャプチャ**: 自動タイマー（OCR）機能の利用時に一時的に使用（画像は保存されません）。

## 🚀 使い方
1. アプリを起動し、初期設定で権限を許可します。
2. 「＋」ボタンから管理したいアプリを登録します。
3. 「設定」から各アプリのロック条件（分/時刻）を決めます。
4. OCRを使いたい場合は、設定画面のアプリ名を長押しして「SS範囲の設定」を行ってください。
